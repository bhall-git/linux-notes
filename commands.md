1. pwd - Print current directory
2. ls - List files
3. cd - Change directory
4. mkdir - Create directory
5. rm - Remove files
6. cp - Copy files
7. mv - Move or rename files
8. cat - Display file contents
9. grep - Search text
10. chmod - Change permissions

## Vim Commands

- i - Insert mode
- Esc - Exit insert mode
- :w - Save
- :q - Quit
- :wq - Save and quit
- :q! - Quit without saving
